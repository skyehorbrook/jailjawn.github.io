Jail Jawn

Description

Jail Jawn creates a graph from Philadelphia's prison population data. The raw data is scraped from the web, provided as "Daily Census" on phila.gov/prisons. The goal of this project is to save this data over time, as the website from which the data is pulled provides no archived data, and provide a visualization to easily understand the data. The data scraping has been automated to retrieve this data daily and store it using Firebase.
